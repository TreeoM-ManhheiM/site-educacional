Situações‑Problema de Projeto e Dimensionamento de Redes

Curso: Desenvolvimento de Sistemas – FATEC
Orientação: Todas as situações abaixo devem ser desenhadas e configuradas manualmente no simulador de redes.

1️⃣ Laboratório de Informática Escolar

Internet → Firewall → Switch
                     |-- PC (15 unidades)
                     |-- Impressora
                     |-- Servidor de arquivos
IP da rede: 192.168.10.0/24
Firewall LAN: 192.168.10.1
PCs: 192.168.10.10–192.168.10.24
Servidor: 192.168.10.100
Impressora: 192.168.10.50
  • Desenhar a arquitetura no simulador
  • Configurar TODOS os IPs fixos
  • Garantir gateway correto em todos os hosts

2️⃣ Sistema Administrativo de Escola

Internet → Firewall → Switch
                     |-- PC (8 unidades)
                     |-- Notebook (2 unidades)
                     |-- Servidor administrativo
Rede: 192.168.20.0/24
Gateway: 192.168.20.1
Servidor: 192.168.20.10
  • Dimensionar rede para crescimento futuro
  • Atribuir IP fixo a servidores e notebooks

3️⃣ Escola com Dois Laboratórios

Internet → Firewall → Switch
           |-- Lab A (10 PCs)
           |-- Lab B (12 PCs)
           |-- Impressora compartilhada
Lab A: 192.168.30.0/24
Lab B: 192.168.40.0/24
Firewall LAN: 192.168.30.1 / 192.168.40.1
  • Definir endereçamento sem conflito
  • Configurar gateways corretamente
