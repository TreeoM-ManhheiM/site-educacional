Plano de Curso – Lógica de Programação

Sequência de aulas utilizada no ensino de programação e desenvolvimento de algoritmos. O objetivo é que o estudante compreenda progressivamente os conceitos fundamentais da programação através de atividades práticas.

  • Fluxogramas
  • Variáveis

]

Semana 21 – Classe


class Aluno:

    def __init__(self,nome):
        self.nome = nome

    def apresentar(self):
        print(self.nome)

a = Aluno("Carlos")
a.apresentar()

Semana 22 – Herança


class Pessoa:
    def falar(self):
        print("Olá")

class Aluno(Pessoa):
    pass

a = Aluno()
a.falar()

Semana 23 – Polimorfismo


class Animal:
    def som(self):
        print("Som")

class Cachorro(Animal):
    def som(self):
        print("Au Au")

c = Cachorro()
c.som()
