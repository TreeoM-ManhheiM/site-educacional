## 📚 Site Educacional – Construção Colaborativa do Conhecimento

Este projeto tem como objetivo promover o **aprendizado ativo em programação, ética na tecnologia e inteligência artificial**, utilizando ferramentas modernas de desenvolvimento e práticas abertas de compartilhamento de conhecimento.

A proposta é integrar **educação, tecnologia e ciência de dados**, permitindo que estudantes experimentem na prática conceitos como:

* lógica de programação
* desenvolvimento web
* coleta e análise de dados
* fundamentos de **Machine Learning**

A atividade interativa presente neste projeto permite que os estudantes respondam questões éticas relacionadas à programação. As respostas podem ser exportadas em **formato CSV**, possibilitando posteriormente análises utilizando ferramentas de **ciência de dados**, como regressão logística em Python.

## 🤖 Inteligência Artificial como apoio educacional

Durante o desenvolvimento deste projeto, ferramentas de **Inteligência Artificial generativa** foram utilizadas como apoio na construção de ideias, estruturação de código e organização pedagógica do material.

Entre elas destaca-se o uso do **ChatGPT**, que auxiliou no processo de:

* geração de exemplos didáticos
* organização de atividades educacionais
* estruturação de código HTML, JavaScript e Python
* reflexão sobre aplicações pedagógicas da tecnologia

A utilização de IA foi realizada como **instrumento de apoio ao processo de construção do conhecimento**, mantendo sempre o protagonismo humano no planejamento educacional.
