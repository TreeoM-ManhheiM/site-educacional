Redes de Computadores e Segurança — Plano Semanal com Exercícios

Cada semana contém 2 exercícios práticos (~50 min cada). Use apenas sua própria máquina e a rede institucional, com ética e responsabilidade.
Observação: Atividades que interagem com a rede devem respeitar as políticas da escola. Evite varrer hosts externos; foque em localhost ou destinos amplamente públicos e permitidos.

Semana 1 — Identificando a Rede Local

Ex1: Descobrir IP, Gateway e DNS

Objetivo: Identificar os principais parâmetros de rede da máquina.
Tempo: 50 minutos.
Recursos: CMD/Terminal, acesso à internet.
Passos:
  1) Abrir o CMD (Windows) e executar: ipconfig /all
     (Linux/macOS: ip addr ou ifconfig)
  2) Anotar: IPv4, Máscara/Prefixo, Gateway padrão e Servidores DNS.
  3) Validar conectividade com ping 8.8.8.8 e ping www.google.com.
  4) Tirar prints da tela e salvar em uma pasta da semana.
Entregáveis: arquivo TXT ou MD com os valores + 2 capturas de tela.
Criterios de avaliação: completude (IP/gateway/DNS), prints e organização.
Observação: Use apenas sua própria máquina e rede da escola.

Ex2: Tabela de Interfaces de Rede

Objetivo: Documentar as interfaces de rede ativas e suas funções.
Tempo: 50 minutos.
Recursos: CMD/Terminal, Python opcional.
Passos:
  1) Listar interfaces: ipconfig (Windows) | ip link / ip addr (Linux/macOS).
  2) Para cada interface ativa, registrar: nome, IPv4/IPv6 (se houver), MAC e status.
  3) (Opcional) Usar Python (psutil/netifaces, se disponível) OU parsing do ipconfig
     para gerar uma tabela simples (CSV ou Markdown).
  4) Salvar como interfaces_semana01.csv ou .md.
Entregáveis: tabela com interfaces e campos solicitados.
Critérios: exatidão dos dados e formatação.

                    ALLOW = {80, 443, 22}
                    import datetime
                    def decide(porta: int):
                        permitido = porta in ALLOW
                        with open("firewall.log", "a", encoding="utf-8") as f:
                            f.write(f"{datetime.datetime.now()} porta={porta} status={'PERMITIDO' if permitido else 'NEGADO'}
")
                        return permitido
                    if __name__ == "__main__":
                        for p in [22, 53, 80, 135, 443]:
                            print(p, '→', 'PERMITIDO' if decide(p) else 'NEGADO')

Entregáveis: script .py e CSV.
Critérios: execução e explicação.
Código base (opcional)
# hash_senhas.py
import hashlib, csv
entradas = ["senha1", "dados_confidenciais", "projeto_redes"]
with open("hashes.csv", "w", newline="", encoding="utf-8") as f:
    w = csv.writer(f, delimiter=';')
    w.writerow(["texto", "sha256"])
    for t in entradas:
        h = hashlib.sha256(t.encode()).hexdigest()
        w.writerow([t, h])
print("Gerado hashes.csv")

# subnet_calc.py
import ipaddress
net = ipaddress.ip_network("192.168.10.0/24")

    hosts = list(sub.hosts())

Ex1: Ping múltiplo (lista de hosts)

Código base (opcional)
# ping_multi.py
import subprocess, csv, sys
hosts = ["www.microsoft.com", "www.github.com", "www.python.org", "www.gov.br", "www.wikipedia.org"]
def avg_ping(host):
    # Windows (-n), Linux/Mac (-c)
    count_flag = "-n" if sys.platform.startswith("win") else "-c"
    p = subprocess.run(["ping", count_flag, "3", host], capture_output=True, text=True)
    out = p.stdout
    # captura simples
    for line in out.splitlines():
        if "Average" in line or "avg" in line or "média" in line.lower():
            return line.strip()
    return "n/d"
with open("ping_resultados.csv", "w", newline="", encoding="utf-8") as f:
    w = csv.writer(f, delimiter=';')
    w.writerow(["host", "resumo"])
    for h in hosts:
        w.writerow([h, avg_ping(h)])

# mini_server_headers.py
from http.server import HTTPServer, SimpleHTTPRequestHandler
class Mini(SimpleHTTPRequestHandler):
    def end_headers(self):
        self.send_header('X-Content-Type-Options', 'nosniff')
        self.send_header('X-Frame-Options', 'DENY')
        self.send_header('Referrer-Policy', 'no-referrer')
        super().end_headers()
HTTPServer(('localhost', 8000), Mini).serve_forever()
