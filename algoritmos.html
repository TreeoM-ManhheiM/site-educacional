Estruturas Condicionais em Python

1️⃣ Desvio Condicional Simples

Executa apenas se a condição for verdadeira.

if idade >= 18:
    print("Você é maior de idade")

2️⃣ Desvio Condicional Composto

Possui duas possibilidades: verdadeiro ou falso.

if idade >= 18:
    print("Entrada permitida")
else:
    print("Entrada proibida")

3️⃣ Desvio Condicional Encadeado

Permite múltiplas decisões usando elif.

if nota >= 7:
    print("Aluno aprovado")

elif nota >= 5:
    print("Aluno em recuperação")

else:
    print("Aluno reprovado")

4️⃣ Operadores Lógicos

AND → ambas condições verdadeiras
OR → apenas uma verdadeira
NOT → nega a condição 

if idade >= 18 and carteira == "sim":
    print("Pode dirigir")

elif idade >= 18 and carteira != "sim":
    print("Maior de idade, mas não pode dirigir")

else:
    print("Menor de idade")

5️⃣ Desvio Condicional Aninhado

Um IF dentro de outro IF.

if idade >= 18:

    if nota >= 7:
        print("Adulto aprovado")

    else:
        print("Adulto reprovado")

6️⃣ Condicional em uma linha (Ternário)

Forma compacta de escrever um if/else.

status = "Maior de idade" if idade >= 18 else "Menor de idade"

print("Status:", status)

7️⃣ Match Case (Switch do Python)

Disponível no Python 3.10 ou superior.

dia = int(input("Digite um número de 1 a 3: "))

match dia:

    case 1:
        print("Segunda-feira")

    case 2:
        print("Terça-feira")

    case 3:
        print("Quarta-feira")

    case _:
        print("Dia inválido")
