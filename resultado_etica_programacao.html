Pipeline de Inteligência Artificial

Os dados coletados na atividade de ética em programação podem ser utilizados para treinar um modelo de Machine Learning usando Regressão Logística.

Fluxo da atividade:

Atividade HTML (respostas dos alunos) ↓ Arquivo CSV com resultados ↓ Python + Pandas ↓ Regressão Logística (Machine Learning) ↓ Modelo treinado para análise dos dados

Exemplo de Código em Python


import pandas as pd
from sklearn.linear_model import LogisticRegression

dados = pd.read_csv("resultado_etica_programacao.csv")

X = dados[["P1","P2","P3","P4","P5","P6","P7"]]
y = dados["Pontuacao"] >= 5

modelo = LogisticRegression()
modelo.fit(X,y)

print("Modelo treinado!")

Nesse exemplo:

  • P1 a P7 representam as respostas dos alunos
  • Pontuação indica o desempenho na atividade
  • O modelo aprende padrões usando Regressão Logística

Esse processo mostra como uma atividade simples pode gerar dados para estudo em Machine Learning.
