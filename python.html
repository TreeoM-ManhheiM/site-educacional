Bibliotecas de Inteligência Artificial em Python

Lista de bibliotecas utilizadas nas disciplinas de Inteligência Artificial e Aprendizado de Máquina.

1. Bibliotecas Fundamentais

NumPy

Operações matemáticas e vetoriais.

pip install numpy

Pandas

Manipulação e análise de dados.

pip install pandas

Matplotlib / Seaborn

Visualização de dados e criação de gráficos.

pip install matplotlib seaborn

2. Aprendizado de Máquina Clássico

Scikit-learn

Biblioteca com algoritmos clássicos de Machine Learning como regressão, classificação, clustering e pré-processamento de dados. 

pip install scikit-learn

3. Deep Learning

TensorFlow

Framework para construção e treinamento de redes neurais.

pip install tensorflow

Keras

Interface de alto nível para criação de redes neurais.

pip install keras

PyTorch

Framework flexível para pesquisa e desenvolvimento de modelos de IA.

pip install torch torchvision torchaudio

4. Boosting e Modelos Avançados

XGBoost

Algoritmo poderoso baseado em boosting utilizado em muitos problemas de ciência de dados.

pip install xgboost

LightGBM

Biblioteca otimizada para treinamento rápido de modelos baseados em árvores.

pip install lightgbm

5. NLP e Modelos Pré-treinados

Transformers (Hugging Face)

Biblioteca utilizada para trabalhar com modelos de linguagem como BERT, GPT e outros modelos avançados de processamento de linguagem natural. 

pip install transformers

6. Computação Numérica Avançada

JAX

Biblioteca para computação numérica acelerada em GPU e TPU, muito utilizada em pesquisas de Inteligência Artificial. 

pip install jax jaxlib
