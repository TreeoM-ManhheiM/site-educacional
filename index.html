Projeto de Coleta de Dados - Aprendizado em Ciência de Dados

Projeto de Coleta de Dados e Inteligência Artificial

Esta atividade faz parte de uma proposta de construção coletiva do conhecimento na área de Ciência de Dados e Desenvolvimento de Sistemas. Os estudantes respondem um formulário digital e o sistema gera automaticamente um arquivo CSV que posteriormente será utilizado para análise de dados e treinamento de modelos de aprendizado de máquina.

Proposta Pedagógica

Este projeto utiliza programação, coleta de dados e análise computacional como forma de aprendizagem ativa. A atividade envolve:

  • Coleta de respostas dos estudantes
  • Geração automática de arquivo CSV
  • Análise dos dados com Python
  • Aplicação de algoritmos de Machine Learning

A proposta valoriza o uso de tecnologias abertas e o compartilhamento de conhecimento, inspirando-se em comunidades como o software livre, projetos colaborativos e plataformas de desenvolvimento como o GitHub.
