Evolução do Código: Laços e Controles

Veja como passamos de um programa simples para fluxos controlados usando laços de repetição, pausas (break) e saltos (continue).

1. Estrutura de Decisão Simples if / else

Este programa lê a idade de apenas um cliente, imprime a tarifa correspondente e encerra a execução. Para consultar o próximo, seria necessário rodar o código de novo.

# 11. Tarifas Básicas
id = int(input("Idade: "))

if id < 6:
    print("Isento")
elif id >= 60:
    print("Meia")
else:
    print("Inteira")
# Fim
Limitação: Imagine um caixa de cinema usando este sistema. O funcionário teria que abrir o programa repetidas vezes para cada pessoa na fila.

2. Estrutura de Repetição Infinita while True / break

Agora o programa entra em um loop infinito. Ele atende de forma contínua e só para quando o comando break é ativado por uma condição (a variável sentinela "s").

# Tarifa Contínua
while True:
    i = int(input("Idade: "))
    
    if i < 6: 
        print("Grátis")
    elif i >= 60: 
        print("Meia")
    else: 
        print("Inteira")
        
    # Condição de parada (Sentinela)
    if input("Sair? (s/n): ") == "s": 
        break # Quebra o laço e finaliza o programa
# Fim
Vantagem: O comando while True mantém o sistema rodando. O comando break dá ao usuário o poder de interromper o loop de forma controlada.

3. Ignorando Dados Inválidos continue

E se o usuário digitar uma idade negativa por engano? O comando continue permite ignorar (pular) o restante do bloco naquela rodada e voltar imediatamente para o início do laço.

# Validação com Continue
while True:
    idade = int(input("Digite a idade (0 para sair): "))
    
    if idade == 0:
        print("Encerrando o sistema...")
        break
        
    if idade < 0:
        print("Erro: Idade não pode ser negativa!")
        continue # Volta para a linha do 'while' na mesma hora
        
    print("Idade de", idade, "anos registrada com sucesso.")
Diferença crucial: O break chuta você para fora do laço. O continue apenas pula a vez atual e recomeça a próxima repetição.

4. Limite de Tentativas (Login) while + break

Aqui o laço já tem uma condição de parada natural (limite de 3 tentativas). Porém, se o usuário acertar a senha antes, usamos o break para encerrar o ciclo antecipadamente.

# Sistema de Login
tentativas = 0

while tentativas < 3:
    senha = input("Digite a senha: ")
    
    if senha == "1234":
        print("Acesso Permitido!")
        break # Sai do loop imediatamente, pois acertou
        
    print("Senha incorreta.")
    tentativas += 1

# Verifica se esgotou as chances
if tentativas == 3:
    print("Conta bloqueada após 3 tentativas.")
Aplicação real: É o padrão usado em caixas eletrônicos e sistemas de login. O loop te dá um limite de chances, mas para de perguntar assim que você acerta.
