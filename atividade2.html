Atividade 5 — Desconto na Cantina

Aluno → 10%. Professor → 15%. Visitante → sem desconto.

Código (Python)

tipo = input('Tipo (aluno/professor/visitante): ').strip().lower()  # entrada: lê texto do usuário; remove espaços extras nas bordas; converte para minúsculas
valor = float(input('Valor da compra (R$): ').replace(',', '.'))  # entrada: lê texto do usuário
if tipo == 'aluno':  # decisão: testa condição e executa o bloco se for verdadeira
    desc = 0.10  # define a porcentagem de desconto para a condição atual
elif tipo == 'professor':  # decisão alternativa: testa nova condição quando o if anterior falha
    desc = 0.15  # define a porcentagem de desconto para a condição atual
else:  # caso contrário: executa quando as condições anteriores falham
    desc = 0.0  # define a porcentagem de desconto para a condição atual
total = valor * (1 - desc)  # calcula o total aplicando o desconto
print(f'Desconto: {desc*100:.0f}% | Total a pagar: R$ {total:.2f}')  # saída: exibe mensagem/resultado na tela
